# 扫码登录

## 运行命令

```
go run cmd/login.go
```

## 目录结构

```
.
├── assets ········· 资源文件
├── cmd ············ 项目主干
├── config ········· 配置文件
├── docs ··········· 用户文档
├── internal
│   ├── app ········ 应用程序代码
│   ├── consts ····· 常量
│   ├── entity ····· 实体
│   ├── models ····· 模型层
│   ├── route ······ 路由文件夹
│   └── service ···· 服务层
└── pkg
    ├── config ····· 解析配置文件
    ├── db ········· 数据库
    ├── logger ····· 日志
    ├── sessions ··· 会话
    └── utils ······ 工具包
```
